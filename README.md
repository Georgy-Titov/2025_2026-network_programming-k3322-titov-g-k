# Сетевое программирование 2025-2026

## Описание

Этот репозиторий посвящён учебной дисциплине "Сетевое программирование", которая является логическим продолжением курса по "Введение в маршрутизацию на предприятии". Курс ориентирован на получение теоретических и практических знаний в области маршрутизации корпоративных IP-сетей, а также на освоение современных технологий организации сетевой инфраструктуры.
